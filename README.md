# Portfolio website

Personal portfolio of Yannick Djallo — a Design Engineer and Creative based in Amsterdam who handles concept, design, and build under one roof. Features seven projects spanning product design, app design, branding, and design engineering. Built with Next.js and Framer Motion.

## Tech stack

- **Next.js 16** (App Router)
- **ShadCN UI** (Radix-based components)
- **Tailwind CSS 4**
- **TypeScript**
